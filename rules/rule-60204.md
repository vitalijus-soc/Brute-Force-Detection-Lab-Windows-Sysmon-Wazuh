# Rule 60204 – Multiple Windows Logon Failures

## Description
This rule triggers when multiple failed login attempts are detected within a short time period.

## Detection logic
- Multiple Event ID 4625 events
- Short time interval
- Same host / account

## Why important
Indicates potential brute-force attack or credential guessing activity.

## MITRE ATT&CK
T1110 – Brute Force
